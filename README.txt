*About the Game

This game is called "Adventure Game". When the game starts, the player has full health and money which is random between 50 to 310. However, the player doesn't have any points. The player wins the game if they reach 10 points and loses if it falls below 0 points.
There are 4 cities in which enemy exists. The player is expected to fight with the enemy in each city. The player can buy the weapons to fight the enemies and armours to reduce the health of the enemy. Moreover, there are healing pads in the shop to increase the player's health by 50.


*Packages/Libraries Used

-Tkinter (For Graphical User Interface)
-Random (For random function)
